Glossary for Microelectronic Systems

links will go to relevant note (imo)

A

  • ADC (Analog-to-Digital Converter): A peripheral that converts continuous real-world analog voltages into discrete digital binary numbers.
  • Address Bus: Wires used by the MCU to specify the physical memory address it wants to read from or write to.
  • ALU (Arithmetic Logic Unit): The part of the CPU core responsible for performing mathematical and logical operations.
  • Anti-aliasing Filter: An analog low-pass filter placed before an ADC to physically block frequencies higher than the Nyquist limit, preventing digitized signal distortion.
  • Asynchronous: Serial communication that does not use a shared clock line. Both devices must pre-agree on a speed (Baud rate).

B

  • Baud Rate: The number of symbol or signal level changes per second in a communication channel.
  • Bit Rate: The number of individual binary bits transmitted per second.
  • Bus: A communication system that transfers data between components inside the MCU (Data, Address, and Peripheral buses).

C

  • Chip Select (CS): A dedicated control wire in SPI used by the Master to wake up and communicate with one specific Slave device.
  • Context Saving: The process where the MCU pauses the main program and pushes its current state onto the stack before executing an ISR.

D

  • DAC (Digital-to-Analog Converter): A peripheral that converts discrete digital binary numbers into a continuous analog voltage.
  • Data Bus: Wires that carry the actual data being processed around the MCU.
  • Duty Cycle: The percentage of time a PWM signal is in the active (HIGH) state during a single period.

E

  • Embedded System: A dedicated computer system designed for a specific control function within a larger mechanical or electrical device.

F

  • Floating Pin: A digital input pin that is not connected to a definite logic voltage (leaving it susceptible to random noise).
  • Flyback Diode: A diode placed in reverse-parallel with an inductive load (like a motor) to safely dissipate inductive kickback voltage and protect the MCU.
  • Full-Duplex: Communication where data can be sent and received simultaneously (e.g., SPI, UART).

H

  • Half-Duplex: Communication where data can flow in both directions, but only one direction at a time (e.g., I2C).
  • Harvard Architecture: A computer architecture with separate, independent memory spaces and buses for data and instructions.

I

  • I2C (Inter-Integrated Circuit): A 2-wire, half-duplex, synchronous serial protocol that uses embedded addressing instead of physical Chip Select lines.
  • Inductive Kickback: A massive, potentially damaging voltage spike created when the current flowing through an inductor (like a DC motor) is suddenly turned off.
  • ISR (Interrupt Service Routine): A specialized block of code that executes immediately when a hardware interrupt is triggered.

L

  • LSB (Least Significant Bit): The bit in a binary number with the lowest mathematical value. In UART, this is typically transmitted first.

M

  • MCU (Microcontroller Unit): A small computer on a single integrated circuit containing a processor core, memory, and programmable I/O peripherals.
  • MSB (Most Significant Bit): The bit in a binary number with the highest mathematical value. In SPI and I2C, this is typically transmitted first.

N

  • Nyquist Theorem: A rule stating that to accurately digitize a signal, you must sample it at least twice as fast as its highest frequency component.

P

  • Parity Bit: An extra bit added to the end of a data frame for simple error checking (e.g., Even Parity ensures the total number of 1s in the frame is an even number).
  • Polling (Software): A technique where the MCU continuously checks the status of an input device over and over in a while loop.
  • Pull-up / Pull-down Resistor: A resistor used to tie a floating pin to a known logic level (HIGH or LOW) when a mechanical switch is open.
  • Pulse Width: The absolute time (usually in milliseconds or microseconds) that a PWM signal is HIGH during one period.
  • PWM (Pulse Width Modulation): A highly efficient technique to simulate an analog voltage by rapidly switching a digital transistor ON and OFF.

Q

  • Quantisation Error: The physical difference between the actual continuous analog input value and its digitized "staircase" representation (Max error = LSB).

R

  • RAM (SRAM): Volatile memory used to store temporary runtime variables. Data is lost when power is removed.
  • Resolution: The smallest physical change in an analog signal that a converter can measure or output (Formula: ).
  • ROM / Flash: Non-volatile memory used to store the compiled program code. Data remains when power is removed.

S

  • SPI (Serial Peripheral Interface): A 4-wire, full-duplex, synchronous serial protocol.
  • Synchronous: Communication that relies on a shared physical Clock line to coordinate data transfer timing.

U

  • UART (Universal Asynchronous Receiver-Transmitter): A hardware device/protocol that handles asynchronous serial communication using just TX and RX wires.

V

  • Von Neumann Architecture: A computer architecture that uses a single, shared memory space and a single bus for both data and instructions.
